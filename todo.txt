сделать отдельные таблицы имен для переменных и для функций в начале компиляции

для переменных:
{
    имя
    адрес в оперативной памяти
    область видимости
    объявлена/не объявлена
}

для функций:
{
    имя
    объявлена/не объявлена
    количество параметров
}

// при вызове функции в стэк кладем все параметры в обратном порядке
// при заходе в функцию записываем в оперативку параметры из стэка
// таким образом гораздо проще передавать как параметры числа и вызовы функций в стэк
